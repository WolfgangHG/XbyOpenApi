# XbyOpenApi
A C# sample for accessing the X/Twitter REST API with a Kiota generated client and OAuth1 / OAuth2.


# How the code works

[Here](doc/Kiota.md) you find a description on how to create a C# client using Kiota.

The client must authorize. This can be done with OAuth1 (see [here](doc/OAuth1.md)) or OAuth2 (see [here](doc/OAuth2.md))
